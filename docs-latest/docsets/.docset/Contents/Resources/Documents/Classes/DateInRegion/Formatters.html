  • useSharedFormatters

    If true this instance of DateInRegion will use formatters shared along calling thread. If false a new date formatter is created automatically and used only by a single DateInRegion Usually you don’t need to create a single formatter for each DateInRegion because this operation is expensive. Unless you need of a particular behaviour you will be happy enough to share a single formatter for each thread.
