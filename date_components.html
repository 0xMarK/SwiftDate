Overview

Working with time components in SwiftDate is pretty easy; both DateInRegion and plain Date implements the same set of properties and function to query for time units or manage dates.

Keep in mind: when using Date objects you are working with a Date instance which is expressed in .defaultRegion.

Initial value of this property is Region.GMT() which define a region located in GMT TimeZone and with current device's Locale and Calendar.

You can change the default region by calling Date static func .setDefaultRegion().
